# Yetl Full Stack Application

This repository contains the Yetl fashion ecommerce stack.

## Backend
- Spring Boot (Java 17)
- H2 SQL database with seeded data
- REST endpoints for products, users, cart, wishlist, checkout, and admin analytics

Run locally:

```bash
cd backend
mvn spring-boot:run
```

## Frontend
- Next.js 14 (App Router)
- Pages for home, collections, bestsellers, search, cart, wishlist, admin, and login

Run locally:

```bash
cd frontend
npm install
npm run dev
```
